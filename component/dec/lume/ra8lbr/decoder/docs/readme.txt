blkdiag.doc
- Word document containing a high-level block diagram of the 
    fixed-point RealAudio 8 (gecko2) decoder

callgraph.doc
- Word document containing a function-by-function call graph of the
    fixed-point RealAudio 8 (gecko2) decoder

cpuusage.xls
- Excel spreadsheet summarizing the simulated CPU usage of the
    fixed-point RealAudio 8 (gecko2) decoder for ARM processors
    (in the ARMulator)

memory.xls
- Excel spreadsheet summarizing the memory usage (ROM and RAM) of the
    fixed-point RealAudio 8 (gecko2) decoder
